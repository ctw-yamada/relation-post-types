== Description ==

WordPress 3.0 and up allow to manage new custom post type. Cool !
This plugin allow to build relations between objects of 2 custom types, very useful for manage related content.

For full info go the [Relation Post Types](http://redmine.beapi.fr/projects/show/relations-post-types) page.

== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. Activate the plugin within you WordPress Administration Backend
3. Go to Settings > Relation Post Types and follow the steps on the [Relation Post Types](http://redmine.beapi.fr/projects/show/relations-post-types) page.
